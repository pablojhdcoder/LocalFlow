## Product Vision

### Problem

Many users want to listen to music offline without relying on streaming subscriptions, unstable connections, or cluttered interfaces.

### Solution

LocalFlow allows users to search for music, extract high-quality audio, store it locally, and play it offline with a clean and modern user experience.

### Core Principles

* Offline-first
* User control over their library
* Minimal and distraction-free UI
* Fast and reliable audio playback
* English-only interface (v1)
